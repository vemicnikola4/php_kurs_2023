LABEL => opis kakvo polje sledi
radio button => ajvise jedan selektovan

FIELDSET tag => Grupiše elemente forme kojoj joj pripadaju u logičku celinu, nad kojom se kasnije mogu vršiti određene akcije

INPUT => najcesci tag polje za unos. input:text npr precica.

TEXTAREA => unos duzeg sadrzaja

SELECT => padajuca lista

CHECKBOX => moze vise opcija

RADIOBUTTON=>najvise jedna selektovana opcija
